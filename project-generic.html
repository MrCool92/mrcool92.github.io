Project XY

I took full responsibility to port Vlambeer's Luftrausers gamemaker build into native c+ + and had two experienced programmers join to help with this huge task. With two fellow programmers we created a framework in c+ + that was easily compiled to other platforms and ported the gamemaker code and assets over to this environment. Big challenges were keeping up with changing design from Vlambeer, getting performance up on the PS Vita, supporting each platforms own social API (Steam, PS3, PS Vita, Gog) and getting through all of the TRCs from Sony. Being a huge task for three people who were also studying and doing internships at the same time as this, it was an incredible educational project, and my first fully shipped cross-platform title!

Features

  • Windows, Linux, MacOSX, Playstation Vita, Playstation 3
  • Own built c+ + framework/engine
  • Social API integrations
  • Working with original developers, publishers and first-party (Stream, GoG and Sony)
